# Gallery（媒体浏览）UI & 交互改进

> 对应 `Views/GalleryView.axaml`，当前 531 行。

## 缩略图网格

### 无虚拟化
`WrapPanel` 一次性渲染全部缩略图，上千张照片时滚动卡顿。

**方案**: 改用 `VirtualizingWrapPanel` 或分页加载（每次 200 张），超出视口的不渲染。

### 缩略图尺寸固定
当前硬编码 `ItemWidth="160" ItemHeight="120"`，无法缩放。

**方案**: 工具栏加 `Slider`，范围 120—240px，实时调整缩略图大小。

### 无搜索/筛选
无法按文件名、标签、日期范围、媒体类型筛选。

**方案**: 照片网格上方加搜索框（`TextBox` + 搜索图标），支持：
- 文件名模糊搜索
- 标签关键词搜索
- 媒体类型筛选（全部 / 照片 / 视频）
- 日期范围快捷筛选（今天 / 本周 / 本月 / 自定义）

### 无大图预览/灯箱
双击 `OpenFileCommand` 调系统默认应用打开，无法在应用内预览。

**方案**: 加全屏预览弹窗（`OverlayPopupHost` 或独立 `Window`），功能：
- 原图全屏显示 + 鼠标滚轮缩放
- 左右方向键 / 屏幕按钮前后翻页
- 显示 EXIF（拍摄日期、曝光、焦距、ISO）
- 幻灯片自动播放
- 右上角显示同步状态和 AI 标签

### 骨架屏简陋
"加载中..." 一行文字，无视觉节奏感。

**方案**: 替换为 shimmer 动画骨架 — 与缩略图同等大小的 `Border` + 渐变闪烁动画，先渲染 20 个占位。

### 多选限于当前分组
"全选" 只作用于当前日期/标签组，无法跨组选择。

**方案**:
- 全局 "全选" 按钮（选中全部可见照片，而非仅当前组）
- Shift + 点击范围选择（在网格内按住 Shift 点击第二张，选中矩形区域内的全部照片）
- 已选计数跨组统计

### 无键盘导航
纯鼠标操作，无方向键支持。

**方案**:
- 方向键在网格内移动焦点，高亮当前所在 tile
- Enter = 打开大图预览
- Space = 选/取消当前
- Ctrl+A = 全选
- Delete = 删除已选
- Esc = 退出多选 / 关闭预览

## 左栏时间轴

### 日期无数量标记
时间轴每个节点只显示日期，看不出哪天有多少张照片。

**方案**: 日期文字后显示数量（如 `2024-07-09 · 42 张`），用 `Text.Secondary` 颜色。

### 日期列表扁平
多年拍摄数据时，滚动极长、无层次感。

**方案**: 改为可折叠的 年 → 月 → 日 三级树形结构：
```
2024
├─ 07月 (156 张)
│  ├─ 07-09 (42 张)
│  └─ 07-08 (35 张)
├─ 06月 (89 张)
...
```

### 时间轴不联动滚动
切换照片网格滚动位置后，左侧时间轴不跟踪当前所在日期。

**方案**: 右侧 `ScrollViewer` 的 `Offset` 变化时，计算当前视口内最多的日期，在左侧高亮对应节点并 `ScrollIntoView`。

## 左栏标签面板

### 标签无筛选
标签多时（100+ 个 AI 标签）难以定位。

**方案**: 标签列表上方加 `TextBox`，输入即过滤（前缀匹配 + 包含匹配）。

### 标签无排序
列表按数据加载顺序，无序排列。

**方案**: 标签列表支持排序切换：
- 按数量降序（默认）
- 按名称字母序
- 按最近使用

## 照片卡片 (photo-tile)

### Hover 反馈弱
仅 `:pointerover` 时加 `BoxShadow`，无放大效果或其它视觉反馈。

**方案**: hover 时加微放大 transform（scale 1.05） + 加亮边框 + 显示完整文件名 tooltip。

### 无加载状态
图片解码慢时，tile 直接显示占位星座图标，无渐现过渡。

**方案**: 图片加载完成后从透明渐现（`Opacity` 0→1 过渡），避免突兀的闪现。

### 视频无时长标记
视频 tile 左上角只显示播放图标，无时长信息。

**方案**: 左下角或右上角叠加时长文字（如 `00:32`），格式 `m:ss`。

## 状态管理

### 滚动位置不记忆
切换日期/标签分组后，再切回来滚动位置丢失。

**方案**: 按分组 key 缓存 `ScrollViewer.Offset`，切回时恢复。
