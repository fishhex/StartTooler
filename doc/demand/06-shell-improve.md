# Navigation & Shell（导航与外壳）UI & 交互改进

> 对应 `Views/MainWindow.axaml`（278 行）、`Controls/NavRail.axaml`、`ViewModels/MainWindowViewModel.cs`（287 行）。

## 窗口与标题栏

### 窗口标题静态
始终显示"星助"，不分页面。

**方案**: 标题栏动态显示当前页面名：
```
星助 — 媒体
星助 — 上传服务
星助 — 垃圾筒
星助 — 设置
```

### 标题栏无窗口控制按钮
macOS 下 `ExtendClientAreaToDecorationsHint="True"`，红绿灯按钮可能在深色背景上不可见。

**方案**: 自定义红绿灯按钮（红/黄/绿），与标题栏风格统一。或至少留出左侧 margin 避免与导航 rail 重叠。

### 标题栏利用率低
标题栏 38px，仅有居中"星助"二字，大片空白。

**方案**: 标题栏左侧（NavRail 上方 80px 区域）放 app logo/图标，中间显示页面信息，右侧可放全局搜索入口或通知铃铛图标。

## NavRail（左侧导航栏）

### 无键盘快捷键
4 个 nav-item 只能鼠标点击。

**方案**: 快捷键：
- `Ctrl+1` = 媒体
- `Ctrl+2` = 上传
- `Ctrl+3` = 垃圾筒
- `Ctrl+4` = 设置

### 无标签/文字
NavRail 只显示图标（如有），没有文字标签，新用户需要悬停才知道按钮功能。

**方案**: 图标下方加小号文字标签（如 `13px`），或 hover 时在右侧弹出浮动 tooltip 说明。

### 无未读/状态标记
垃圾筒按钮无数量标记，不知垃圾筒是否有内容。

**方案**: 垃圾筒 nav-item 右上角加数量徽章（红点 + 数字），空筒时隐藏。

### 无最小化/紧凑模式
80px 宽的 NavRail 在小屏幕上占比较大。

**方案**: 窗口宽度 < 900px 时自动折叠为 48px（仅图标），或提供手动折叠按钮。

## 工具栏

### 按钮组布局拥挤
工具栏左侧按钮一排展开，多选模式下"多选/取消多选/全选/反选/批量上传/批量下载/释放本地/删除/开始AI"一字排开，加上进度文字，小窗口下溢出。

**方案**:
- 正常模式：只显示核心操作（多选、排序）
- 多选模式：将操作分组，用下拉 `SplitButton` 收纳"全选/反选"为二级菜单
- 批量操作（上传/下载/释放/删除）合并为"批量操作 ▾" SplitButton
- 窗口窄时自动折叠溢出项到"更多 ▾"菜单

### 排序选择器无图标
`ComboBox` 只有"时间↓"/"评分↓"文字，视觉效果弱。

**方案**: 排序图标 + 文字（`↑↓` 排序图标 + "时间" / "评分"），下拉时显示当前选中项前有 ✓。

## 全局导航

### "返回"按钮交互弱
设置页和垃圾筒页左上角"返回"按钮，样式是 `secondary-button`（灰色描边），不太显眼。

**方案**: 返回按钮加左箭头图标（`←`），或放在标题栏左侧区域取代居中标题。

### 页面切换无过渡动画
`ContentControl.Content` 直接切换，无渐变或滑动效果。

**方案**: 加 `PageSlide` 过渡动画（240ms ease-in-out）：
- 设置/上传/垃圾筒：从右侧滑入
- 返回媒体页：从左侧滑入

### 无面包屑导航
从媒体页进入设置后，不知道自己在哪一层。

**方案**: 标题栏或工具栏加面包屑（如 `媒体 > 设置 > OSS配置`），可点击跳回任意层级。

## 状态栏

### 无状态栏
缺少全局信息展示。

**方案**: 窗口底部加 32px 状态栏，内容：
- 左侧：总文件数 / 已上传数（如 `📁 1,234 张 / ☁️ 892 已上传`）
- 中间：上次刷新时间（如 `🕐 3 分钟前刷新`）
- 右侧：OSS 连接状态圆点（绿/红）
- 右侧：网络状态（在线/离线）
- 右侧：磁盘可用空间（如 `💾 45.2 GB 可用`）

## 快捷键

### 无全局快捷键
整个应用纯鼠标驱动。

**方案**:
| 快捷键 | 操作 |
|--------|------|
| `Ctrl+1/2/3/4` | 切换 NavRail 页面 |
| `Ctrl+R` | 刷新扫描 |
| `Ctrl+A` | 全选（多选模式） |
| `Ctrl+Shift+A` | 取消全选 |
| `Ctrl+I` | 反选 |
| `Ctrl+D` | 删除已选 |
| `Delete` | 删除（需确认） |
| `Escape` | 退出多选 / 关闭预览 / 返回上一页 |
| `F5` | 刷新 |
| `Ctrl+F` | 聚焦搜索框 |
| `Ctrl+,` | 打开设置 |
